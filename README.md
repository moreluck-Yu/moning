# Moning 项目索引

## 项目概述

Moning 是一个智能化的个人习惯养成系统，通过 AI 驱动的多模态内容生成和多平台分发，实现早起打卡和单词学习的数字化仪式感。

### 核心设计原则
1. **优雅降级** - AI生成 → API匹配 → 静态备选的多层保障
2. **多模态融合** - 文字 + 图片 + 音频的沉浸式体验
3. **认知外化** - 将个人习惯数字化为可检索的知识资产
